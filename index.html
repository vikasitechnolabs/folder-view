⌨️ Shortcuts Guide / शॉर्टकट गाइड

→ Arrow Right → Next Photo (अगली फोटो)

← Arrow Left → Previous Photo (पिछली फोटो)

K / Space → Keep / Select (फोटो चुनें)

D → Reject (फोटो हटाएं)

Backspace → Undo (पिछला एक्शन वापस लें)

Mouse / माउस:

Click side images → Navigate (नेविगेट करें)

Tip: Use keyboard for fastest workflow 🚀
सुझाव: तेज काम के लिए कीबोर्ड इस्तेमाल करें
